Código fonte / Pré Formatado

Caso eu queira um texto que facilita a leitura, pois dessa maniera está cada letra de um tamanho.
Basta eu usar a tag code E o texto irá ficar dessa maneira, mais fácil para ler.

Também posso usar a tag pre. Ela basicamente ira aparecer da mesma maneira que eu escrevi no código,
ou seja ira considerar tudo, espaços quebras de linhas entre outras... 

Esse texto esa exatamente da mesma maneira
        que eu escrevi no código!

Citações Simples

Caso eu queira fazer uma citação ao invés de eu usar manualmente a aspas, é só eu usar a tag q no texto que eu quero citar. Um exemplo:
O amigo do meu pai disse: o computador é apenas um burro mais rápido
Para usar a aspas manualmente, é só em caso de querer dizer algo sem sentido do tipo isso é "mais ou menos"interessante.
Nesse caso eu usei manualmente

Citações Completas

Caso eu queria fazer uma citaçao mais completa, basta usar a tag blockquote e inserir o texto dentro dela dessa maneira:
Segundo a autora Robertha Pereira:

A linguagem HTML não reconhece os caracteres de quebra de linha dos editores de texto e, mesmo que exista uma linha em branco no código fonte, os clientes Web só reconhecem o início de um novo parágrafo mediante à marcação apropriada
E para inserir o link do artigo ou livro, é so colocar dentro da tag blockquote a tag cite e colar o link. Esse link não aparece no site, mas na parte interna isso aparecerá.
